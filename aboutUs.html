Somos apenas dois estudantes de Engenharia Informática que querem ter uma boa nota a ITW. Para este projeto, tivemos de pesquisar muito sobre os Jogos Olímpicos e aprender a trabalhar com Knockout (totalmente por vontade própria).

Este projeto é um site informativo sobre os Jogos Olímpicos. No fundo, este trabalho não é nada mais nada menos do que uma demonstração das skills que adquirimos nesta disciplina. Feito com muito amor e carinho (e algumas lágrimas), é um website que tem muito para oferecer. Estejam à vontade para explorá-lo!

Tal como é necessário utilizar diferentes linguagens para desenvolver sites (Html, Javascript, CSS...), também achamos pertinente recorrer a diferentes línguas para testar os vossas capacidades bilinguísticas de interpretação).
